smartwed – AI-Powered Wedding Planning App
SmartWed is a comprehensive, AI-powered wedding planning mobile application developed as a Final Year Project (FYP). It provides couples, vendors, and admins with a centralized platform to manage wedding tasks, budgets, bookings, invitations, playlists, and real-time assistance through an AI chatbot.
The app is built using Flutter and Dart for cross-platform mobile development and Firebase for authentication, real-time database, and cloud storage, ensuring scalability, security, and real-time updates.
_Project Overview_
Planning a wedding is complex
 involving multiple tasks, vendors, budgets, and timelines.
 SmartWed simplifies this process by providing:
Role-based access for Couples, Vendors, and Admin
AI-powered chatbot for instant guidance and support
Centralized dashboards for tracking tasks, budgets, bookings, and invitations
A secure and interactive platform for all wedding stakeholders
_Objectives_
